# Skill-14: 笔记模块规范

## 前置依赖
skill-03, skill-09

## 目标
定义笔记列表页、笔记编辑页的完整行为规范。

---

## 1. 独立笔记列表页

### 1.1 页面结构
```
┌─────────────────────────────────────────┐
│ ← 笔记                      [🔍] [FAB+] │
├─────────────────────────────────────────┤
│ 笔记标题1        2024-01-15 14:30       │  ← 笔记列表项
│ 内容预览前两行...                        │
├─────────────────────────────────────────┤
│ 笔记标题2        2024-01-14 10:00       │
│ 内容预览2前两行...                       │
├─────────────────────────────────────────┤
│ 关联笔记标题3    关联: IMG_001.jpg       │  ← 关联笔记显示媒体名
│ 内容预览3前两行...                       │
└─────────────────────────────────────────┘
```

### 1.2 列表项显示
- 标题（TitleMedium，截断单行）
- 更新时间（LabelSmall，灰色）
- 内容前 2 行预览（BodyMedium，灰色）
- 关联媒体的文件名（仅关联笔记，LabelSmall，Primary 色）

### 1.3 行为
- 点击：打开笔记编辑页（覆盖层）
- 长按：弹出删除菜单
- 列表按 updatedAt 降序排列
- 显示全部笔记（独立 + 关联）

### 1.4 空状态
- 图标 + "还没有创建笔记" + "点击 + 按钮创建新笔记"

---

## 2. 笔记编辑页

### 2.1 页面结构
```
┌─────────────────────────────────────────┐
│ ← 编辑笔记 / 新建笔记      [保存] [⋮]   │
├─────────────────────────────────────────┤
│ 标题                                     │  ← 标题输入框
│ ─────────────────────────────────────── │
│                                         │
│ 内容                                     │  ← 内容输入框
│                                         │
│                                         │
│                                         │
└─────────────────────────────────────────┘
```

### 2.2 标题输入框
- 单行文本输入
- 占位文字："输入标题..."
- 无字数限制

### 2.3 内容输入框
- 多行文本输入
- 占位文字："输入笔记内容..."
- 支持换行
- 无字数限制
- 自动换行

### 2.4 保存规则
- 点击"保存"按钮保存
- 标题和内容都为空时不保存（提示"标题和内容不能都为空"）
- 新建：创建 Note 实体并插入数据库
- 编辑：更新现有 Note 的 title、content、updatedAt
- 保存后返回上一页

### 2.5 未保存离开确认
- 用户修改了标题或内容后点击返回
- 弹出确认对话框："有未保存的更改，确定要离开吗？"
- "离开"：放弃更改，返回上一页
- "继续编辑"：取消离开

### 2.6 更多菜单
| 操作 | 说明 |
|------|------|
| 关联媒体 | 显示当前关联的媒体（如果有），点击跳转到媒体详情 |
| 删除 | 确认后删除笔记并返回上一页 |

### 2.7 关联媒体显示
- 如果笔记有关联媒体（mediaId != null），在内容区下方显示关联媒体卡片
- 卡片显示缩略图和文件名
- 点击卡片跳转到媒体详情页

## 3. 验证标准

- [ ] 独立笔记列表正确显示（独立 + 关联笔记）
- [ ] 列表项信息完整（标题、时间、预览、关联媒体名）
- [ ] 笔记编辑页标题和内容输入正确
- [ ] 保存功能正确（新建和编辑）
- [ ] 空标题和内容不保存
- [ ] 未保存离开确认正确
- [ ] 关联媒体显示和跳转正确
- [ ] 删除笔记正确执行
- [ ] 空状态正确显示
