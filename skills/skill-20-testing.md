## 3. 手动测试场景

### 3.1 首次启动流程
1. 安装应用 → 启动
2. 验证权限请求对话框
3. 授权 → 进入首页（空状态）
4. 验证设置默认值

### 3.2 媒体导入流程
1. 点击 FAB → 从设备导入
2. 浏览文件系统 → 选择文件
3. 确认导入 → 观察进度
4. 验证导入结果（成功/跳过/失败）
5. 验证缩略图生成
6. 验证数据库记录

### 3.3 完整 CRUD 测试
1. 创建相册 → 添加媒体 → 编辑相册 → 删除相册
2. 创建标签 → 打标签 → 编辑标签 → 删除标签
3. 创建笔记 → 编辑笔记 → 删除笔记
4. 验证所有级联删除

### 3.4 多选操作测试
1. 长按进入多选 → 选择多个媒体
2. 批量添加到相册 → 验证结果
3. 批量打标签 → 验证结果
4. 批量删除 → 验证结果
5. 全选 → 验证计数

### 3.5 搜索测试
1. 搜索存在的媒体 → 验证结果
2. 搜索存在的笔记 → 验证结果
3. 搜索不存在的关键词 → 验证空状态
4. 验证防抖行为
5. 验证关键词高亮

### 3.6 设置测试
1. 切换主题模式 → 验证即时生效
2. 切换语言 → 验证提示重启
3. 切换网格列数 → 验证即时生效
4. 清除全部数据 → 验证完全清理
