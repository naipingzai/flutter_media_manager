## 2. "所有媒体"Tab — 完整规范

### 2.1 页面结构

```
┌─────────────────────────────────────────┐
│  [全部] [图片] [视频]  │ [全部▾] [FAB+] │  ← 过滤器行 + FAB
├─────────────────────────────────────────┤
│ ┌───┐ ┌───┐ ┌───┐                      │
│ │   │ │   │ │   │                       │  ← 媒体网格
│ └───┘ └───┘ └───┘                       │
│ ┌───┐ ┌───┐ ┌───┐                      │
│ │   │ │   │ │   │                       │
│ └───┘ └───┘ └───┘                       │
│ ...                                      │
└─────────────────────────────────────────┘
```

### 2.2 媒体类型过滤器（FilterChip 行）

| 过滤器 | 显示文字 | 逻辑 |
|--------|---------|------|
| 全部 | "全部" | 显示所有已导入媒体 |
| 图片 | "图片" | 只显示 type = IMAGE 的媒体 |
| 视频 | "视频" | 只显示 type = VIDEO 的媒体 |

**行为**：
- 默认选中"全部"
- 切换过滤器时，网格内容淡出 → 淡入新内容
- 过滤器状态在 Tab 切换时保持

### 2.3 内容过滤器（下拉菜单）

第二个 FilterChip 显示当前过滤模式名称（默认"全部"），点击展开下拉菜单：

| 选项 | 对应 FilterMode | 说明 |
|------|----------------|------|
| 全部 | ALL | 显示全部媒体 |
| 有标签的 | WITH_TAGS | 仅显示已打标签的媒体 |
| 无标签的 | WITHOUT_TAGS | 仅显示未打标签的媒体 |
| 有相册的 | WITH_ALBUMS | 仅显示已加入相册的媒体 |
| 无相册的 | WITHOUT_ALBUMS | 仅显示未加入相册的媒体 |

**行为**：
- 默认选中"全部"
- 切换时网格内容淡出 → 淡入新内容
- 过滤器状态在 Tab 切换时保持

### 2.4 媒体网格

#### 网格布局
- 使用 LazyVerticalGrid
- 列数由设置中的 gridColumns 决定（默认 3 列）
- 列间距和行间距均为 SpacingXxs（2dp）

#### 网格项结构（预览模式，showContentPreview = true）

```
┌─────────────────┐
│                 │  ← 缩略图（正方形，铺满）
│    缩略图        │
│                 │
│             🎬  │  ← 视频时长徽章（仅视频）
│  ✓              │  ← 选中勾选图标（多选模式）
├─────────────────┤
│ IMG_001.jpg     │  ← 文件名（BodySmall，截断显示）
│ 2.5 MB          │  ← 文件大小（LabelSmall，灰色）
└─────────────────┘
```

#### 网格项结构（非预览模式，showContentPreview = false）

```
┌─────────────────┐
│                 │
│    [图标]        │  ← 媒体类型图标（大尺寸占位）
│                 │
├─────────────────┤
│ IMG_001.jpg     │  ← 文件名
│ 2.5 MB          │  ← 文件大小
└─────────────────┘
```

#### 视频时长徽章
- 位置：缩略图右下角
- 格式：mm:ss（如 02:30）
- 背景：半透明黑色圆角矩形
- 文字：白色 LabelSmall

#### 选中态
- 缩略图左上角显示 Primary 色勾选图标
- 整个缩略图覆盖 30% 透明度的 Primary 色
- 缩略图显示 3dp Primary 色边框

### 2.5 过滤器切换行为

- 切换过滤器（图片/视频 FilterChip 或内容过滤器）时：
  1. 如果当前处于多选模式 → **自动清空所有选中项**，退出多选模式
  2. 当前网格内容淡出（200ms）
  3. 执行对应查询
  4. 新网格内容淡入（220ms）
  5. 无结果时显示对应空状态文案
- 过滤器状态在 Tab 切换时保持（切换到"相册"Tab 再切回来，过滤器仍然是之前选中的）
- 多选模式下，过滤器行**隐藏**（不占空间），退出多选模式后恢复显示

### 2.6 空状态

当"所有媒体"没有媒体时：

```
┌─────────────────────────────────────────┐
│                                         │
│              [📂 图标]                   │
│          还没有导入任何媒体               │
│       点击 + 按钮开始导入                │
│                                         │
└─────────────────────────────────────────┘
```

- 居中显示空状态图标（64dp）
- 主提示文字："还没有导入任何媒体"
- 副提示文字："点击 + 按钮开始导入"

### 2.6 FAB（浮动操作按钮）

| 状态 | 外观 | 行为 |
|------|------|------|
| 正常模式 | 蓝色圆形 "+" 图标 | 点击展开菜单 |
| 多选模式 | 隐藏 | 不显示 FAB |
| 菜单展开 | 弹出两个选项 | 见下方 |

**FAB 菜单选项**：

| 选项 | 图标 | 文字 | 行为 |
|------|------|------|------|
| 从设备导入 | 文件夹图标 | "从设备导入" | 打开文件浏览器 |
| 选择目录导入 | 目录图标 | "选择目录导入" | 打开目录选择器（选择整个目录一次性导入） |

### 2.7 多选模式

#### 进入多选模式
- 长按任意媒体项 → 进入多选模式，该项自动选中

#### 多选模式下的 UI 变化

| 组件 | 变化 |
|------|------|
| TopAppBar | 切换为多选模式 TopAppBar（返回 + 计数 + 全选 + 取消） |
| 底部导航栏 | 隐藏，替换为多选底栏 |
| FAB | 隐藏 |
| 网格项 | 全部显示选中态覆盖层（未选中的显示空心圆圈，已选中的显示实心勾选） |

#### 多选底栏

```
┌─────────────────────────────────────────┐
│  [添加到相册]  [添加标签]  [删除]        │
└─────────────────────────────────────────┘
```

| 按钮 | 说明 |
|------|------|
| 添加到相册 | 打开相册选择对话框（多选模式），将选中媒体批量添加到相册 |
| 添加标签 | 打开标签选择对话框（多选模式），给选中媒体批量打标签 |
| 删除 | 弹出确认对话框，确认后批量删除选中媒体 |

#### 多选操作

| 操作 | 行为 | 完成后 |
|------|------|--------|
| 点击网格项 | 切换该项选中状态 | 更新计数 |
| 全选 | 选中当前过滤器下的所有媒体 | 更新计数 |
| 取消 | 退出多选模式 | 恢复正常 UI |
| 添加到相册 | 弹出相册选择对话框 | 完成后退出多选模式 |
| 添加标签 | 弹出标签选择对话框 | 完成后退出多选模式 |
| 删除 | 弹出确认对话框 | 确认后删除并退出多选模式 |

#### "添加到相册"对话框（P10 完整规范）

多选模式下点击"添加到相册"，弹出相册选择对话框：

```
┌─────────────────────────────────────┐
│ 添加到相册（已选 3 个文件）           │
├─────────────────────────────────────┤
│ ┌────┐ ┌────┐ ┌────┐               │
│ │封面│ │封面│ │封面│               │
│ ├────┤ ├────┤ ├────┤               │
│ │旅行│ │工作│ │美食│               │  ← 顶层相册
│ └────┘ └────┘ └────┘               │
│ ┌────┐ ┌────┐                       │
│ │封面│ │封面│                       │
│ ├────┤ ├────┤                       │
│ │↳   │ │↳   │                       │  ← 子相册，前缀 "↳"
│ │三亚│ │北京│                       │
│ └────┘ └────┘                       │
│ ┌────┐                               │
│ │封面│                               │
│ ├────┤                               │
│ │↳   │                               │  ← 更深层子相册
│ │海边│                               │     breadcrumb: "旅行 > 三亚 > 海边"
│ └────┘                               │
└─────────────────────────────────────┘
           [取消]
```

**行为规范**：
- 对话框标题显示已选文件数量："添加到相册（已选 N 个文件）"
- 显示**所有层级**的相册，按深度排序（顶层在前，子层在后）
- 同层内按 sortOrder 升序排列
- 子相册名称前显示 "↳" 前缀标识
- 如果存在同名相册（不同层级），通过 breadcrumb 路径区分（如 "旅行 > 三亚" vs "工作 > 三亚"）
- 封面显示规则：有封面媒体用缩略图，无封面但有媒体用第一张缩略图，无媒体显示灰色背景+空相册图标
- 无相册时显示"暂无相册"提示
- 点击相册卡片即完成添加，自动关闭对话框
- **不提供新建相册入口**（新建相册在相册 Tab 的 FAB 中操作）
- 点击"取消"关闭对话框
- 添加完成后自动退出多选模式（清空选择、恢复底部导航栏、恢复 FAB）

### 2.8 文件浏览器（全屏覆盖层）

文件浏览器在"所有媒体"Tab 内作为全屏覆盖层显示，具体行为参见 skill-05。

打开方式：FAB → 从设备导入

### 2.9 媒体网格点击行为

| 操作 | 正常模式 | 多选模式 |
|------|---------|---------|
| 单击 | 打开媒体详情页（覆盖层） | 切换选中状态 |
| 长按 | 进入多选模式并选中该项 | 无额外行为 |

### 3.4 相册创建（FAB）

在"相册"Tab 中，FAB 行为变为创建新相册：

```
用户点击 [+新建] FAB
    → 弹出对话框：
        ┌─────────────────────────────┐
        │ 创建相册                      │
        ├─────────────────────────────┤
        │ 名称: [输入框____________]    │
        │                              │
        │ 父相册: [下拉选择 ▼]         │  ← 可选，不选则为顶层相册
        │   · （无，顶层相册）          │
        │   · 旅行                     │
        │   ·   └ 2024年               │  ← 子相册缩进显示
        │   ·   └ 2023年               │
        │   · 工作                     │
        │   · 美食                     │
        ├─────────────────────────────┤
        │         [取消]  [创建]       │
        └─────────────────────────────┘
```

**行为规范**：
- 点击 FAB 弹出"创建相册"对话框
- 名称输入框：文本输入，限制 50 字符，不允许为空
- 父相册下拉选择器：
  - 默认选项："（无，顶层相册）"
  - 列出所有已有相册（扁平列表，子相册缩进显示，前加 "└" 符号）
  - 选择某个相册作为父相册 → 新相册成为该相册的子相册
  - 不选择任何父相册 → 新相册为顶层相册（parentId = null）
- 点击"创建"：校验名称非空 → 创建相册 → 关闭对话框 → 刷新列表
- 点击"取消"：关闭对话框，不创建
- 创建后相册出现在列表中正确位置（顶层或对应父相册下）

### 4.2 标签列表项

每个标签项显示：
- 颜色圆点（如果标签有颜色）或默认灰色圆点
- 标签名称（TitleMedium）
- 媒体数量（BodySmall，灰色，括号内）
- 子标签缩进显示（24dp 缩进）

### 4.3 标签列表行为

| 操作 | 行为 |
|------|------|
| 点击标签 | 打开标签媒体列表页（覆盖层） |
| 长按标签 | 弹出编辑/删除菜单 |
| 展开/折叠 | 子标签可折叠/展开 |

### 4.4 标签创建（FAB）

在"标签"Tab 中，FAB 行为变为创建新标签：

| 步骤 | 行为 |
|------|------|
| 点击 FAB | 弹出"创建标签"对话框 |
| 输入名称 | 文本输入框，限制 30 字符 |
| 选择颜色 | 颜色选择器（可选） |
| 选择父标签 | 标签选择器（可选，不选则为顶层标签） |
| 确认 | 创建标签 |
| 取消 | 关闭对话框 |
