# Skill-12: 标签模块规范

## 前置依赖
skill-03, skill-09

## 目标
定义标签媒体列表页、标签 CRUD、标签选择器的完整行为规范。

---

## 1. 标签媒体列表页

### 1.1 页面结构

```
┌─────────────────────────────────────────┐
│ ← 标签名称           [编辑] [删除] [🔍] │  ← TopAppBar
├─────────────────────────────────────────┤
│ 🏷 子标签1  🏷 子标签2                   │  ← 子标签行（可选）
├─────────────────────────────────────────┤
│ ┌───┐ ┌───┐ ┌───┐                      │
│ │   │ │   │ │   │                       │  ← 媒体网格
│ └───┘ └───┘ └───┘                       │
└─────────────────────────────────────────┘
```

### 1.2 子标签行
- 横向滚动的 Chip 列表
- 点击子标签跳转到子标签的媒体列表（覆盖层叠加）
- 无子标签时不显示此区域

### 1.3 媒体网格
- 与首页媒体网格相同布局和行为
- 支持多选模式
- 点击打开媒体详情

### 1.4 空状态
- 居中图标 + "此标签下暂无媒体"

---

## 2. 标签 CRUD 操作

### 2.1 创建标签
- 名称输入（必填，限制 30 字符）
- 颜色选择（可选，预设 12 种颜色 + 无颜色）
- 父标签选择（可选，不选则为顶层标签）

### 2.2 编辑标签
- 可修改名称、颜色、父标签
- 不允许将标签设为自身的子标签（循环检测）

### 2.3 删除标签
- 确认对话框："确定要删除标签「{name}」吗？关联的媒体不会被删除。"
- 级联删除所有子标签
- 级联删除所有 MediaTag 关联记录
- 不删除媒体文件本身

---

## 3. 标签选择器对话框

### 3.1 单媒体模式
- 从媒体详情页的标签面板触发
- 显示所有标签列表，已关联的标签显示选中态
- 支持搜索过滤
- 点击切换关联状态（即时生效）
- 可直接创建新标签

### 3.2 多选模式
- 从首页多选底栏"添加标签"触发
- 显示所有标签列表
- 支持搜索过滤
- 选择后点击"确认"批量应用到所有选中媒体
- 可直接创建新标签

### 3.3 搜索过滤
- 对话框顶部有搜索输入框
- 输入时实时过滤标签列表
- 支持按标签名模糊匹配

## 4. 预设颜色与颜色选择器

### 4.1 12 种预设颜色

| # | 色值 | 名称 | 对应 MaterialColor |
|---|------|------|-------------------|
| 1 | #F44336 | 红色 | Red |
| 2 | #FF9800 | 橙色 | Orange |
| 3 | #FFC107 | 黄色 | Amber |
| 4 | #4CAF50 | 绿色 | Green |
| 5 | #2196F3 | 蓝色 | Blue |
| 6 | #673AB7 | 紫色 | DeepPurple |
| 7 | #E91E63 | 粉色 | Pink |
| 8 | #607D8B | 蓝灰 | BlueGray |
| 9 | #00BCD4 | 青色 | Cyan |
| 10 | #8BC34A | 浅绿 | LightGreen |
| 11 | #795548 | 棕色 | Brown |
| 12 | #9E9E9E | 灰色 | Gray |

### 4.2 颜色选择器 UI 规范

```
颜色:  ●  ●  ●  ●  ●  ●  ●  ●  ●  ●  ●  ●
       红  橙  黄  绿  蓝  紫  粉  灰 青 浅绿 棕 灰
```

- 12 个圆点水平排列，可换行（每行 6 个）
- 圆点尺寸：28dp × 28dp
- 圆点间距：8dp
- 选中的圆点：外圈加 2dp 白色边框 + 内圈显示 16dp 的勾选图标（白色）
- 未选中的圆点：纯色圆点
- 可以不选颜色（标签 color = null，显示为默认灰色圆点）

### 4.3 创建/编辑标签对话框完整规范

```
┌─────────────────────────────┐
│ 创建标签 / 编辑标签           │
├─────────────────────────────┤
│ 名称: [输入框____________]    │
│                              │
│ 颜色: ● ● ● ● ● ●           │
│       红 橙 黄 绿 蓝 紫      │
│       ● ● ● ● ● ●           │
│       粉 灰 青 浅绿 棕 灰    │
│                              │
│ 父标签: [下拉选择 ▼]         │
│   · （无，顶层标签）          │
│   · 旅行                     │
│   ·   └ 国内                 │  ← 子标签缩进显示
│   ·   └ 国外                 │
│   · 美食                     │
│   · 工作                     │
├─────────────────────────────┤
│         [取消]  [创建]       │
└─────────────────────────────┘
```

**行为规范**：
- 编辑模式下，禁止将标签设为自身的子标签或子孙标签（循环检测）
- 编辑模式下，"创建"按钮文字变为"保存"
- 父标签下拉列表中，当前标签自身及其所有子孙标签**不显示**（防止循环引用）
- 创建后标签立即出现在标签列表中正确位置
