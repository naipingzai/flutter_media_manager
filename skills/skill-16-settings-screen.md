# Skill-16: 设置页面规范

## 前置依赖
skill-03, skill-08, skill-09

## 目标
定义设置页面的 UI 结构和交互行为。

---

## 1. 页面结构

```
┌─────────────────────────────────────────┐
│ ← 设置                                 │  ← TopAppBar
├─────────────────────────────────────────┤
│ 外观                                     │  ← 分组标题
│   主题模式        跟随系统 ▾             │  ← 下拉选择
│   动态颜色        [开关]                 │  ← Switch
├─────────────────────────────────────────┤
│ 语言                                     │
│   语言           跟随系统 ▾              │
├─────────────────────────────────────────┤
│ 显示                                     │
│   网格列数        3 ▾                   │
│   显示内容预览    [开关]                 │
├─────────────────────────────────────────┤
│ 存储                                     │
│   清除所有数据    [按钮]                 │
├─────────────────────────────────────────┤
│ 关于                                     │
│   版本            1.0.0                  │  ← 只读
└─────────────────────────────────────────┘
```

---

## 2. 设置项交互

| 设置项 | 交互组件 | 选项 |
|--------|---------|------|
| 主题模式 | 下拉菜单 | 跟随系统/浅色/深色 |
| 动态颜色 | Switch 开关 | 开/关 |
| 语言 | 下拉菜单 | 跟随系统/中文/English |
| 网格列数 | 下拉菜单 | 2/3/4/5 |
| 显示内容预览 | Switch 开关 | 开/关 |

---

## 3. 变更行为

| 设置项 | 即时生效 | 延迟生效 |
|--------|---------|---------|
| 主题模式 | ✅ 即时切换 | — |
| 动态颜色 | ✅ 即时切换 | — |
| 语言 | — | 需要重建 Activity，提示"语言设置将在重启后生效" |
| 网格列数 | ✅ 即时生效 | — |
| 显示内容预览 | ✅ 即时生效 | — |

---

## 4. 清除所有数据

- 点击按钮弹出确认对话框
- 确认文案："确定要清除所有数据吗？此操作将删除所有媒体、笔记、相册和标签，且不可撤销。"
- 确认后执行：
  1. 删除数据库所有表数据
  2. 删除应用私有存储中所有媒体文件
  3. 删除 thumbnails 目录
  4. 重置所有设置为默认值
- 完成后显示"所有数据已清除"Toast
- 不退出应用，显示空状态首页
