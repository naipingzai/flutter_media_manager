# Skill-13: 媒体详情页规范

## 前置依赖
skill-03, skill-09

## 目标
定义媒体详情页的媒体预览、信息面板、笔记面板、标签面板的完整行为规范。

## 2. 媒体预览区

### 2.1 图片预览
- 使用 Coil 加载图片
- 支持双指缩放（1x ~ 5x）
- 支持拖拽平移（缩放状态下）
- 双击切换 1x ↔ 2x 缩放
- 加载失败显示占位图

### 2.2 视频预览
- 显示视频第一帧作为封面
- 中央显示播放按钮
- 点击播放按钮在原位播放视频
- 显示播放/暂停、进度条、时长
- 点击视频区域同样跳转到全屏 MediaViewerActivity 进行全屏播放

### 2.3 点击预览区 → 跳转全屏查看器

- **点击预览区**或**点击 TopAppBar 右侧 [全屏] 按钮** → 打开 MediaViewerActivity（独立全屏 Activity）
- 跳转动画：淡入淡出
- 传递参数：
  - `mediaId`：当前媒体 ID
  - `mediaList`：当前上下文的媒体 ID 列表（如相册内、标签内、首页所有媒体）
  - `currentIndex`：当前媒体在列表中的索引
- 全屏查看器中可左右滑动切换上/下一张媒体
- 全屏查看器中显示操作栏（分享、删除、添加标签、添加笔记），参见 skill-17
- 全屏查看器中删除媒体后，返回详情页时详情页需刷新或自动关闭（如列表为空）
- 从全屏查看器返回时，详情页预览区同步更新（如缩略图可能已变化）

## 3. 媒体信息面板

### 3.1 折叠/展开
- 默认展开
- 点击标题行折叠/展开
- 折叠时只显示标题和箭头图标

### 3.2 显示字段

| 字段 | 说明 | 条件 |
|------|------|------|
| 类型 | 文件 MIME type | 始终显示 |
| 尺寸 | 宽×高（像素） | 有宽高信息时显示 |
| 大小 | 格式化文件大小 | 始终显示 |
| 导入时间 | 格式化日期时间 | 始终显示 |
| 时长 | mm:ss 格式 | 仅视频 |
| 文件路径 | 完整路径 | 始终显示 |

### 3.3 信息布局
- 采用两列网格布局
- 左列：字段名（BodySmall，灰色）
- 右列：字段值（BodyMedium）

## 4. 笔记面板

### 4.1 折叠/展开
- 默认展开
- 点击标题行折叠/展开

### 4.2 笔记列表
- 每条笔记显示：
  - 标题（TitleSmall）
  - 更新时间（LabelSmall，灰色）
  - 内容前 2 行预览（BodyMedium，灰色）
- 点击笔记项打开笔记编辑页（覆盖层）
- 列表按 updatedAt 降序排列

### 4.3 新建笔记
- 点击"+ 新建"按钮
- 打开笔记编辑页（覆盖层），自动关联当前媒体
- 新建的笔记 mediaId 设为当前媒体 ID

### 4.4 笔记面板空状态
- 无笔记时显示"暂无笔记"
- 显示"+ 新建"按钮

### 4.5 笔记操作
| 操作 | 行为 |
|------|------|
| 点击笔记 | 打开编辑页（覆盖层） |
| 长按笔记 | 弹出删除菜单 |
| 删除笔记 | 确认对话框 → 删除 |

---

## 5. 标签面板

### 5.1 折叠/展开
- 默认展开
- 点击标题行折叠/展开

### 5.2 标签 Chip 列表
- 横向流式布局（FlowRow）
- 每个标签显示为 Chip：
  - 标签名称
  - 颜色圆点（如果有颜色）
  - 点击跳转到该标签的媒体列表（覆盖层）
  - 长按或点击"×"移除标签

### 5.3 添加标签
- 点击"+ 添加"按钮
- 打开标签选择器对话框（单媒体模式）
- 参见 skill-12 §3.1

### 5.4 移除标签
- 点击标签 Chip 上的"×"按钮
- 立即移除（无需确认）
- 标签本身不被删除，仅移除关联关系

### 5.5 标签面板空状态
- 无标签时显示"暂无标签"
- 显示"+ 添加"按钮

---

## 6. TopAppBar 操作

| 操作 | 说明 |
|------|------|
| 全屏按钮 | 打开 MediaViewerActivity 全屏查看 |
| 更多菜单 | 展开菜单：分享、删除 |

### 6.1 分享
- 调用系统分享 Intent
- 分享原始文件

### 6.2 删除
- 确认对话框："确定要删除此媒体吗？此操作不可撤销。"
- 确认后：
  - 删除数据库记录
  - 删除应用私有目录中的文件
  - 删除缩略图
  - 删除所有关联的笔记、标签关联、相册关联
  - 返回上一页
